Лабораторна робота №1

Виконайте наступну послідовність:

1) Ознайомтеся з акаунтом сайту primat.org на github: https://github.com/primats.

2) Зареєструйте свій github акаунт, ознайомтеся з функціоналом і основними можливостями, створіть сторінку read.me.

3) Створіть свій перший репозитарій, синхронізуйте відповідній папці на вашому комп'ютері.

4) Створіть вашу персональну сторінку: сайт-візитку, який буде доступний за адресою vashnik.github.io.

5) Налаштуйте синхронізацію файлів сайту з вашої папки на комп'ютері, встановіть Git Shell для візуалізації роботи по синхронізації.

6) Відредагуйте первинний стандартний шаблон github, встановіть на вашому сайті посилання на сайт математика.укр, primat.org.

7) Ознайомтеся з технологією Gist, опублікуйте там приклад коду, розмістіть його на вашому сайті vashnik.github.io, використовуючи код Gist для вбудовування на сторінках сайту.

8) Виконайте мікро-розмітку у відповідності з вимогами schema.org сторінок сайту і проведіть валідацію для Google (https://search.google.com/structured-data/testing-tool).

9) Знайти і усунути всі помилки на сторінках вашого сайту. Приклад, пошуку помилок для сайту кафедри ().

Лабораторна робота №2

Завдання1. Побудуйте семантичну мережу (не менш як трирівневу), яка відтворює відносини для наступного об’єкту знань чи алгоритму (у відповідності до вашого варіанту за номером у журналі):

1) Структурно-логічна (семантична) схема навчальних предметів за курсами, що вивчаються студентами кафедри та кафедрами, що їх викладають.

2) Структура навчальних підрозділів та служб університету.

3) Семантична мережа приміщень кафедри.

4) Семантична мережа спеціальностей університету, за якими навчаються студенти КрНУ.

5) Семантична мережа вибору спеціальності для навчання в університеті (за уподобанням абітурієнта).

Завдання 2. Побудуйте графічне зображення семантичної мережі, використовуючи спеціалізовані засоби для розміщення на html-сторінці, наприклад, технології svg-графіки чи інші.

Лабораторна робота №3

На вашому персональному сайті vashnik.github.io розмістити окремо по одному на кожній сторінці, візуалізацію даних, виконану у вигляді графіка, онлайн діаграми і карти. Для візуалізації використовувати технології Morris Charts, jQuery Knob, Flot, jVectorMap.

Варіанти завдань:

1) Кількість студентів спеціальності кафедри по роках для різних курсів;

2) Демографічна ситуація в Кременчуці по роках;

3) Кількісні показники по факультету Фекі по роках (число студентів, розподіл по кафедрам);

4) Результати ЗНО по школах Кременчука. Порівняльний аналіз на графіках;

5) Порівняння результатів вступної компанії за фахом кафедри з аналогічними кафедрами області та регіону;

6) Аналіз середніх балів успішності студентів кафедри по роках (інформація в деканаті);

7) Спортивні досягнення факультету по року і видів спорту.
